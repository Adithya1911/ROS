Steps to Install ROS Noetic on Ubuntu

    Setup the System to Accept ROS Packages:

    Open the terminal and run the following command to set up your system to accept software from the ROS package repository:

    sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'

    Setup Keys and Install curl:

    You need to install the curl package and add the ROS key to your system. If curl is not already installed, use the following commands:

    sudo apt install curl

    curl -s https://raw.githubusercontent.com/ros/rosdistro/master/ros.asc | sudo apt-key add -

    Update the System:

    After setting up the keys, update your system's package index to reflect the changes:

    sudo apt update

    Install ROS:

    Depending on your needs, you can install different variations of ROS Noetic:

        Full Desktop Version (including 2D/3D perception):

        sudo apt install ros-noetic-desktop-full

        Desktop Version (without 2D/3D simulations):

        sudo apt install ros-noetic-desktop

        Base Version (minimal installation with core ROS tools):

        sudo apt install ros-noetic-ros-base

    Install Additional ROS Packages:

    To install any additional ROS packages, use the following command, replacing package_name with the desired package:

    sudo apt install ros-noetic-package_name

    Setup Environment:

    After installing ROS, you need to set up the environment to run ROS commands in your terminal:

        To source the ROS setup file for the current session:

        source /opt/ros/noetic/setup.bash

        To make this setup automatic every time you open a terminal, add the source command to your ~/.bashrc file:

        echo "source /opt/ros/noetic/setup.bash" >> ~/.bashrc

        source ~/.bashrc

    Install Dependencies for Building Projects:

    Finally, install the necessary dependencies for creating projects or workspaces in ROS:

    sudo apt install python3-rosdep python3-rosinstall python3-rosinstall-generator python3-wstool build-essential
